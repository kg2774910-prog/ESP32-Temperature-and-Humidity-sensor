# Temperature and Humidity Sensor using Wokwi

## 🔗 Simulation Link
https://wokwi.com/projects/462827613823862785
## 📝 Description
This project measures temperature and humidity using sensors in Wokwi simulation.

## ▶️ How to Run
1. Open simulation link
2. Click Start Simulation

## 🛠️ Components Used
- ESP32 / Arduino
- DHT Sensor
- Resistor
